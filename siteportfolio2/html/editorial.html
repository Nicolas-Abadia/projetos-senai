Dom Casmurro

Este projeto é uma nova versão do livro Dom Casmurro de Machado de Assis voltada para o público adolescente que frequentemente utiliza o livro Dom Casmurro para vestibulares e provas. Esta nova edição do livro busca ser mais atrativa para quem lê livros digitais, sendo diagramada diretamente pare este meio. A ilustração da capa foi feita digitalmente trazendo um novo ponto de vista para a obra e para o personagem principal da trama. A identidade visual do livro remete ao Brasil século XIX a partir de elementos visuais, ilustrações e tipografias com estética simples e uma fonte confortável para leitura.
